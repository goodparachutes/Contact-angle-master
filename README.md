# Contact Angle Master

面向润湿与表面表征的**接触角时序数据分析工具**。将仪器或后处理导出的 Excel 接触角序列自动分段为静态角（SCA）、前进角（ACA）、后退角（RCA），并计算接触角滞后（CAH），支持人工修正、批量样品对比与报表导出。

## 技术栈

- **前端**：React 19、Vite 8、Tailwind CSS  
- **图表**：Recharts  
- **表格**：通过 SheetJS（CDN）在浏览器中读取 `.xlsx` / `.xls` / `.csv` 首表数据  

## 数据输入说明

- 默认从工作表**第一行表头之下的数据行**读取数值。  
- 接触角数值所在列由配置项「数据列标」指定（默认列 **H**，支持如 `A`、`AA` 等 Excel 列字母）。  
- 每一行视为**按时间或针位移顺序**的一个测量点；可一次导入多个文件形成样品队列。

## 核心算法流程

整体管线与界面内「算法指南」流程图一致：**数据导入 → 物理预过滤 → 中位数窗口分段 → IQR 去噪 → 统计汇总**。

### 1. 物理预过滤（0° 屏蔽）

- 将 **≤ 1.0°** 的读数视为无效/占位，标记为非激活点，不参与后续分段判定与统计，避免拟合失败或零点干扰滑动窗口。

### 2. 阶段分段：SCA → ACA → RCA

在「静态点数」`N`（默认 3）之后，对**激活**点按顺序推断当前属于 SCA、ACA 还是 RCA：

- **SCA**：序列前 `N` 个有效（激活）点强制标为静态接触角段；其后状态切换为 ACA。  
- **ACA / RCA 切换**（中位数窗口）：对每个激活点，取其后 **`windowSize`** 个激活点的接触角中位数 `futureMed`，取其前最多 **`windowSize`** 个激活点的中位数 `pastMed`（代码中需至少一侧凑满窗口长度才参与判定）。  
  - 当前为 **ACA** 时：若 `pastMed - futureMed > angleThreshold` 且当前点相对未来窗口不过大跳变（`pt.value - futureMed < 0.8 × angleThreshold`），则判定进入 **RCA**（后退段）。  
  - 当前为 **RCA** 时：若 `futureMed - pastMed > angleThreshold` 且未来窗口相对当前点满足类似约束，则切回 **ACA**，并**递增循环编号**（用于区分多圈加退湿过程）。  
- **默认参数**：`angleThreshold = 5°`，`windowSize = 3`（可在源码 `config` 中调整；界面主要暴露窗口大小等项）。

使用中位数而非单点值，可减弱单帧尖峰对阶段边界的误判。

### 3. 人工覆盖与排除

- **右键菜单**：可将某点强制设为 ACA 或 RCA 段的**起点**（覆盖自动推断），或标记/取消「噪点」排除。  
- **框选模式**：矩形框选多点后，可批量设为 ACA、批量设为 RCA，或批量切换噪点标记。  
- 被手动指定类型的点会参与分段状态传播；**手动标记的点在 IQR 去噪中豁免**（不会被自动标为离群）。

### 4. IQR 离群检测（可选）

当「自动过滤」开启且某段（SCA / ACA / RCA）激活点数 **≥ 5** 时：

- 对该段激活点接触角排序，取第一、第三四分位数 `Q1`、`Q3`，`IQR = Q3 - Q1`。  
- 容忍系数默认为 **`outlierSensitivity`（默认 4.5）**：有效区间为 `[Q1 - k·IQR, Q3 + k·IQR]`，区间外且非手动的点标为离群。  
- **可调**：去噪容忍度滑块、最小/最大合理角度范围（范围外强制为噪点）、可选「区域外强制噪点」（仅保留指定索引区间内的点）。

### 5. 统计量与接触角滞后（CAH）

对每一样品，在排除离群与噪点规则后：

- **SCA / ACA / RCA**：各段分别计算**算术平均**与**总体标准差**（对段内有效点）。  
- **CAH（接触角滞后）**：对每个**循环编号**，在该圈内分别对 ACA、RCA 有效点求平均，**滞后 = mean(ACA) − mean(RCA)**；若存在多圈，再对多圈的滞后值求平均与标准差，得到汇总的 CAH 与不确定度。

## 主要功能

| 功能 | 说明 |
|------|------|
| 多样品队列 | 多文件导入、切换当前样品、删除样品 |
| 测量曲线 | 折线图展示分段着色；视图缩放与疏密采样显示 |
| 交互修正 | 右键分段/噪点；框选 + Ctrl/Cmd 追加选择 |
| 汇总对比 | 多样品柱状图对比 SCA/ACA/RCA/CAH，带误差条；可勾选参与汇总的样品、拖拽排序 |
| 全屏汇总 | 大屏查看汇总图 |
| 报表导出 | Excel（`Batch_CA_Report.xlsx`），可选导出列与小数位数 |
| 元数据 | 样品显示名、测试液体（含预设 datalist） |
| 外观 | 多套配色预设与自定义颜色 |

应用内 **BookOpen** 按钮可打开「Contact Angle Master 算法指南」弹层，与上述逻辑一致。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

## 免责声明与使用建议

本工具依据**时序上接触角变化趋势**做启发式分段与统计，适用于典型的加液–退液或类似过程数据。不同仪器导出格式、采样密度或物理过程差异较大时，请结合**右键修正、框选批量标注与参数调优**核对结果；重要结论建议与原始曲线及实验记录交叉验证。
